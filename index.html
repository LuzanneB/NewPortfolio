  • add close9/26/2019 Not everything requires a function

    The bootcamp I just completed focuses so much on Javascript my initial approach to almost everything is to write a function, especially when a click event is involved. Today as I was writing an onClick function to toggle the open and close icons for this collapsible, when it dawned on me that there must be a better way. The JQuery that I had put together (similar to what my team used when we put together Food Decoder, linked in the Code section of this page) was slow on the page and was going to require even more code to make it run properly. “It’s just an icon”, I thought.

    After a very quick google search, I found that my initial instinct to write a function was overcomplicating it. The same way I was tapping into the “active” class on the li for a JQuery function to add and remove the class for the corresponding icons, I could put in some super simple CSS to display or not display. What else am I overcomplicating?! The more I learn about CSS, the more magical, simple and fast I find that it is.

  • add close10/1/2019 Documentation can be confusing, videos can be long, patience it is a virtue.

    I confess, documentation sometimes reads like another language to me. I’m a newbie and I don’t always understand the vocabulary. The bootcamp experience allowed me to hit the fast-forward button on many topics in favor of actionable learning. This is good and bad. The good thing is, I can quickly create simple full stack applications. The bad thing is, I missed out on a lot of “101”. Enter google, my handy teacher.

    Today, I purchased luzanneburt.com in order to publish this site. I started reading the docs on namecheap and on github in order to learn how to connect my github and started noting all of the pieces of the pie that I had no clue about so that I could read about that too. “This list is never ending and I want to get this done today”, I thought. I put the reading aside and went to youtube, after a few minutes I found a quick video and within 2 minutes, luzanneburt.com was up and running.

    As I continue to learn new things, I realize there are a ton of things I don’t know but am empowered by the things I do. I’ll take it one step at a time, hitting the fast forward button when I can while ensuring to take time out to learn more of the “101”.
